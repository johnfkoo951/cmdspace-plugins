# APA 7th Edition Quick Reference

## In-Text Citations

### Author-Date System

| 저자 수 | 첫 인용 | 이후 인용 |
|---------|---------|----------|
| 1 | (Kim, 2024) | (Kim, 2024) |
| 2 | (Kim & Lee, 2024) | (Kim & Lee, 2024) |
| 3+ | (Kim et al., 2024) | (Kim et al., 2024) |

### Citation Formats

```
Narrative: Kim (2024) found that...
Parenthetical: ...as demonstrated (Kim, 2024).
Direct quote: "exact words" (Kim, 2024, p. 15).
Multiple sources: (Kim, 2024; Lee, 2023; Park et al., 2022)
Multiple by same author: (Kim, 2023, 2024)
Same surname: (J. Kim, 2024; S. Kim, 2023)
```

## Reference List Format

### Journal Article
```
Author, A. A., & Author, B. B. (Year). Title of article. *Title of Periodical*, *Volume*(Issue), Page-Page. https://doi.org/xxxxx
```

Example:
```
Kim, Y. H., & Lee, S. J. (2024). Knowledge sharing in AI-enhanced workplaces: A systematic review. *Human Resource Development Review*, *23*(2), 145-170. https://doi.org/10.1177/15344843231234567
```

### Book
```
Author, A. A. (Year). *Title of work: Capital letter also for subtitle*. Publisher. https://doi.org/xxxxx
```

### Book Chapter
```
Author, A. A. (Year). Title of chapter. In E. E. Editor (Ed.), *Title of book* (pp. xx-xx). Publisher. https://doi.org/xxxxx
```

### Conference Paper
```
Author, A. A. (Year, Month Day-Day). Title of paper [Paper presentation]. Conference Name, Location. https://doi.org/xxxxx
```

### Dissertation/Thesis
```
Author, A. A. (Year). *Title of dissertation/thesis* [Doctoral dissertation/Master's thesis, University Name]. Database Name.
```

### Website
```
Author, A. A. (Year, Month Day). Title of page. Site Name. https://url
```

## Reference List Rules

1. **Alphabetical order** by first author's surname
2. **Hanging indent**: First line flush left, subsequent lines indented 0.5 inch
3. **DOI always included** when available (as hyperlink)
4. **Italics** for: journal name, volume number, book title
5. **No "Retrieved from"** unless content may change (e.g., social media)
6. **Up to 20 authors** listed; 21+ use first 19 ... last author

## Korean Journal Article Format (한국어)

```
김영호, 이수진 (2024). AI 기반 직장에서의 지식공유: 체계적 문헌 고찰. *한국인력개발학회지*, *26*(2), 145-170. https://doi.org/xxxxx
```

Rules for Korean references:
- Korean author names: 성+이름 without comma (김영호)
- Korean journal names in Korean
- Mix Korean and English references alphabetically (Korean by 가나다, then English by ABC)

## Formatting Specifications

| Element | APA 7th Specification |
|---------|----------------------|
| Font | 12pt Times New Roman or 11pt Calibri |
| Spacing | Double-spaced throughout |
| Margins | 1 inch (2.54 cm) all sides |
| Page numbers | Top right, starting from title page |
| Heading 1 | Centered, Bold, Title Case |
| Heading 2 | Flush Left, Bold, Title Case |
| Heading 3 | Flush Left, Bold Italic, Title Case |
| Heading 4 | Indented, Bold, Title Case, ending with period |
| Heading 5 | Indented, Bold Italic, Title Case, ending with period |

## Abstract

- Separate page after title page
- Label: "Abstract" (centered, bold)
- Single paragraph, no indentation
- 150-250 words (check journal requirements)
- Keywords below abstract: "Keywords: *word1*, *word2*, *word3*"

## Tables

- Numbered sequentially: Table 1, Table 2
- Title: Bold, italic, below table number
- Notes below table: *Note.* or *a* superscript
- No vertical lines
- Horizontal lines: top, below header, bottom only

## Figures

- Numbered sequentially: Figure 1, Figure 2
- Title: Bold, italic, below figure number
- Caption: Plain text below title
